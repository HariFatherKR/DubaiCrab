# 🧠 OpenKlaw - Brainstorming Session

> OpenClaw 경량화 버전 - 한국 사무직을 위한 로컬 AI 비서

---

## 💡 핵심 아이디어

**OpenClaw + 로컬 LLM = 한국 사무직 맞춤 AI 비서**

기업 보안 정책으로 ChatGPT 못 쓰는 직장인들,
개인 노트북에서 돌아가는 가벼운 AI 비서

---

## 🎯 Problem Statement

### 현재 상황

| 문제 | 설명 |
|------|------|
| 🔒 **보안 정책** | 대기업/공공기관 80%+ ChatGPT 사용 금지 |
| 💰 **비용 부담** | Claude/GPT API 월 $20+ 개인 부담 |
| 🖥️ **사양 제한** | 사무용 노트북: i5, 8-16GB RAM, 내장 GPU |
| 🇰🇷 **한국어 특화** | HWP, 공문서, 보고서 양식 이해 필요 |
| 🌐 **오프라인 필요** | 보안망, 출장, 인터넷 불안정 환경 |

### 타겟 사용자

```
👔 "김과장" 페르소나

- 대기업/공공기관 사무직 (30-45세)
- 노트북: LG그램/삼성갤럭시북 (i5, 16GB, 내장GPU)
- 업무: 보고서, 이메일, 회의록, 데이터 정리
- 고민: "ChatGPT 쓰고 싶은데 회사에서 막아놨어..."
- 니즈: 로컬에서 돌아가는 AI, 회사 자료 안 유출
```

### 시장 규모

- 한국 사무직 인구: ~800만명
- ChatGPT 차단 기업 비율: ~60-80%
- 잠재 TAM: 400만+ 사용자

## 🔍 경쟁 분석

### 현재 대안

| 솔루션 | 장점 | 단점 |
|--------|------|------|
| ChatGPT/Claude | 성능 최고 | 회사에서 차단, 유료 |
| Ollama | 로컬 실행 | CLI만, 사무용 특화 X |
| LM Studio | GUI 있음 | 통합 도구 없음 |
| 회사 제공 AI | 보안 OK | 기능 제한, 느림 |
| 엑셀 Copilot | MS 통합 | 라이센스 필요, 비쌈 |

### OpenKlaw 포지셔닝

```
               고성능
                 ↑
                 │    ChatGPT/Claude
                 │         ●
                 │
     로컬 ←──────┼───────→ 클라우드
                 │
          ● OpenKlaw
          │      │    
    Ollama●      │
                 │
                 ↓
               경량
```

**OpenKlaw 포지션**: 로컬 + 중간 성능 + 사무 특화

## 🛠️ 핵심 기능 (Feature Brainstorm)

### Must Have (P0)

| 기능 | 설명 | 왜 필수? |
|------|------|----------|
| **로컬 LLM 실행** | Ollama/llama.cpp 기반 | 보안, 오프라인 |
| **HWP 파싱** | 한글 문서 읽기/요약 | 한국 사무 필수 |
| **이메일 작성** | 비즈니스 이메일 초안 | 일상 업무 |
| **회의록 정리** | 녹음 → 요약 | 시간 절약 |
| **보고서 초안** | 템플릿 기반 작성 | 반복 업무 |

### Should Have (P1)

| 기능 | 설명 |
|------|------|
| **엑셀 분석** | CSV/XLSX 데이터 요약 |
| **일정 관리** | 로컬 캘린더 연동 |
| **문서 검색** | 로컬 파일 시맨틱 검색 |
| **번역** | 영↔한 문서 번역 |
| **PPT 요약** | 슬라이드 → 텍스트 |

### Nice to Have (P2)

| 기능 | 설명 |
|------|------|
| **음성 인터페이스** | Whisper 로컬 |
| **브라우저 확장** | 웹페이지 요약 |
| **Outlook 플러그인** | 이메일 앱 연동 |
| **RAG** | 사내 문서 Q&A |

**추천**: **Ollama** (OpenClaw와 동일)

## 🎯 MVP 범위

### Phase 1: CLI MVP (2주)

```
OpenKlaw v0.1

✅ Ollama 연동 (Qwen2.5-3B)
✅ CLI 인터페이스
✅ HWP 파일 읽기/요약
✅ 이메일 초안 작성
✅ 간단한 Q&A

$ openklaw chat "이 보고서 요약해줘" --file report.hwp
$ openklaw email --tone formal --topic "회의 일정 조율"
```

## 📊 성공 메트릭

### North Star: 주간 활성 사용 시간 (WAT)

> 사용자가 OpenKlaw를 통해 업무를 처리한 시간

### L1 메트릭

| 메트릭 | 타겟 | 측정 |
|--------|------|------|

| NPS | 40+ | 인앱 설문 |

### 핵심 사용 시나리오 (검증용)

3. **"회의록 정리"** - 1시간 → 10분
