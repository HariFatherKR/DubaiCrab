# OpenKlaw UI/UX 리서치 문서

## 1. PM 대시보드 UI/UX 베스트 프랙티스

### 1.1 정보 계층 구조

**시각적 계층 원칙:**
1. **Primary Zone (40%)**: 가장 중요한 KPI/메트릭 - 시선이 가장 먼저 가는 곳
2. **Secondary Zone (35%)**: 상세 데이터 테이블, 차트
3. **Tertiary Zone (25%)**: 부가 정보, CTA, 네비게이션

**적용 방안:**
- 프로젝트 진행률 → Primary (상단 전체 너비)
- 통계 카드 4개 → Primary (바로 아래)
- 연동 도구/기능 목록 → Secondary (2열 그리드)
- 의사결정 로그/CTA → Tertiary (하단)

### 1.2 카드 배치 원칙

**F-패턴 읽기 흐름:**
```
→ → → → → → → (첫 번째 수평 스캔)
↓
→ → → → (두 번째 수평 스캔)
↓
↓
↓ (수직 스캔)
```

**Z-패턴 (간단한 레이아웃):**
```
1 → → → → 2
    ↘
      ↘
3 ← ← ← ← 4
```

**권장사항:**
- 중요 정보는 좌측 상단
- CTA 버튼은 우측 또는 하단 우측
- 관련 정보는 그룹핑

### 1.3 색상 심리학

| 색상 | 심리적 효과 | 사용처 |
|------|------------|--------|
| **틸/민트** | 신뢰, 균형, 성장 | 메인 브랜드 컬러 |
| **그린** | 성공, 완료, 긍정 | 완료 상태, 성공 메트릭 |
| **블루** | 안정, 전문성 | 링크, 정보성 요소 |
| **오렌지/앰버** | 주의, 경고, 에너지 | 진행중, 경고 상태 |
| **레드** | 긴급, 중요 | 에러, 차단 상태 |

### 1.4 네비게이션 패턴

**사이드바 베스트 프랙티스:**
1. 고정 너비 (200-280px)
2. 접기/펼치기 기능
3. 아이콘 + 레이블 조합
4. 현재 위치 명확한 하이라이트
5. 그룹핑 (구분선 또는 섹션 헤더)

## 2. SaaS 대시보드 벤치마킹

### 2.1 Notion
- **특징**: 미니멀, 화이트 스페이스 활용
- **사이드바**: 계층적 네비게이션, 폴더 구조
- **컬러**: 화이트 배경, 블랙 텍스트, 악센트 컬러 최소화
- **시사점**: 콘텐츠 중심, 깔끔한 타이포그래피

### 2.2 Linear
- **특징**: 모던, 다크/라이트 모드
- **사이드바**: 아이콘 중심, 컴팩트
- **컬러**: 퍼플 악센트, 그라데이션 사용
- **시사점**: 키보드 단축키 강조, 빠른 네비게이션

### 2.3 Asana
- **특징**: 컬러풀, 친근한 UX
- **사이드바**: 프로젝트별 컬러 코딩
- **컬러**: 다양한 프로젝트 컬러
- **시사점**: 시각적 구분, 협업 강조

### 2.4 Monday.com
- **특징**: 대담한 컬러, 카드 기반
- **사이드바**: 워크스페이스 기반 구조
- **컬러**: 상태별 색상 (그린, 오렌지, 레드 등)
- **시사점**: 상태 시각화, 직관적 진행률

## 3. 글래스모피즘 + 라이트 테마 트렌드

### 3.1 CoachPro 스타일 분석

**배경:**
```css
/* 소프트 민트/틸 그라데이션 */
background: linear-gradient(
  135deg,
  #e6f5f3 0%,    /* 라이트 민트 */
  #d9ece9 25%,   /* 소프트 틸 */
  #cce4e7 50%,   /* 아쿠아 블루 */
  #c5e1ec 75%,   /* 스카이 블루 */
  #d4eaf5 100%   /* 라이트 블루 */
);
```

**카드 (Glassmorphism):**
```css
background: rgba(255, 255, 255, 0.6);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.8);
border-radius: 20px;
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.08);
```

**텍스트:**
```css
/* Primary */
color: #1e293b;     /* Slate 800 */

/* Secondary */
color: #64748b;     /* Slate 500 */

/* Muted */
color: #94a3b8;     /* Slate 400 */
```

### 3.3 그라데이션 조합

**추천 조합:**
1. **민트 → 틸 → 블루**: 신뢰감, 전문성
2. **민트 → 그린**: 성장, 생산성
3. **틸 → 퍼플**: 창의성, 혁신

## 4. 직관적인 메뉴 배치

### 4.1 기능 우선순위 배치

**상단 (자주 사용):**
1. 📊 Dashboard - 전체 현황
2. 💬 채팅 - 핵심 기능

**중단 (보조 기능):**
3. 📁 프로젝트
4. 📧 이메일
5. 📄 문서

**하단 (설정/기타):**
6. ⚙️ 설정
7. ❓ 도움말

### 4.2 아이콘 + 레이블 조합

**권장 패턴:**
```
[아이콘] [레이블]
  📊      Dashboard
  💬      채팅
  ⚙️      설정
```

**아이콘 가이드라인:**
- 크기: 20-24px
- 간격: 아이콘과 레이블 사이 12px
- 일관성: 같은 스타일 (outline/filled)
- 명확성: 직관적으로 기능 파악 가능

### 4.3 반응형 사이드바

**데스크탑 (≥1024px):**
- 펼침 상태 (220px)
- 아이콘 + 레이블

**태블릿 (768-1023px):**
- 접힘 상태 (70px)
- 아이콘만 표시
- 호버 시 툴팁

**모바일 (<768px):**
- 숨김 상태
- 햄버거 메뉴로 토글
